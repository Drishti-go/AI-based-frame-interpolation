AI based Frame Interpolation Model

Goal:
> Auto-generate videos from WMS satellite imagery using AI-based frame interpolation and display them on Open Layers/Leaflet maps.

Key Features:
> Smooth interpolation between regular intervals (e.g., every minute).
> Handles challenges like deformable objects (e.g., clouds).

Outcome:
> Compatible with OGC WMS, overlays videos on Open Layers (v6+), with optional GPU/NPU utilization.
